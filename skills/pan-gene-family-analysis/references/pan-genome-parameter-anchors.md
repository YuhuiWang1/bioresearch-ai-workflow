# 文献锚定的默认参数表 / Literature-anchored parameter anchors

本文件用于把你筛选的 **5–10 篇高质量植物泛基因组家族文章**的关键参数,提炼成"有文献
背书的可辩护默认值",供本技能各步骤取默认时引用。

> **填充方式(第二轮)**:你提供文献后,助手会**用自己的话归纳每篇的参数(不照搬原文)**,
> 标注出处(物种 / 期刊 / 年份),逐行填入下表。这样既尊重版权,又让默认值可追溯。

## 怎么用 / How to use

各步骤取默认参数时,先查此表是否有同类研究的取值;有则采用文献多数做法并注明,没有则
用 SKILL.md / 各 references 里的通用默认,并标注"可调、待文献校准"。
