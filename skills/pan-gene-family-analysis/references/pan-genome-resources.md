# 泛基因组资源与数据库 / Pan-genome resources

承接 SKILL.md「参考基因组与数据库选择」。本文件用于:开工前先查该物种是否已有可复用
的泛基因组资源,避免重头构建。

## 使用原则 / How to use

1. **先查该物种是否已有公开 pan-genome 项目 / 数据库**:若已有(含种质集合、PAV、SV 图谱、
   非冗余泛基因集),**优先直接复用**其数据与坐标系,只在其上做家族层面的提取与分析。
2. **物种专用数据库优先于综合库**(承接 `gene-family-analysis`):专用库更贴近社区共识、
   更新更及时;综合库用于交叉核对。
3. **记录**:数据库名、pan-genome 项目/版本、种质数量与清单、访问日期,写进 Methods。
