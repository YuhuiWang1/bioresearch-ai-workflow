## 2. 选用建议 / Selection guidance
- **常规家族成员**:先用综合预测器(TargetP / WoLF PSORT / Plant-mPLoc / DeepLoc)交叉验证;
  植物材料优先 Plant-mPLoc、TargetP;拟南芥可加 AtSubP2 / SUBA4。
- **针对特定细胞器的问题**(如"是否定位到叶绿体/线粒体/过氧化物酶体"):改用对应的专一
  工具(ChloroP/PCLR、MitoProt、PredPlantPTS1)以获更可靠结论。
- **已知基因**:用 UniProtKB / SUBA4 做交叉验证或作为"金标准"参照。
- **注意工具自带的已知局限**(如 WoLF PSORT 对分泌蛋白较弱、CELLO/PSLpred 偏细菌)。

## 3. 理解预测结果:分选信号与多重定位 / Interpreting predictions

**预测只是辅助,不能当作事实判断**——生信预测基于已有数据和算法,各工具都有适用范围与
限制,结果只能作为后续实验的研究思路与参考。最终定位需实验确认。

**分选信号原理**(决定蛋白去哪个区室):蛋白在细胞质中合成后,由分选信号引导转运。分选
信号分两类——
- **信号序列 Signal sequence**:包括导肽(N 端)和信号肽(可在肽链任意位置),通常
  15–60 个连续氨基酸残基,定向转移后可能被信号肽酶切除。→ 这正是 TargetP / iPSORT /
  ChloroP 等**基于序列**的预测器所识别的部分。
- **信号斑 Signal patch**:由肽链不同部位几段序列折叠后形成的三维斑块,完成分选后仍存在。
  → 属**三维结构**信息,序列预测器往往捕捉不到,是预测可能失准的原因之一。

**多重定位(关键,常被误读为"冲突")**:一个蛋白定位到 ≥2 个区室非常常见——据报道约
**60% 的蛋白存在多重定位**,部分活跃蛋白可在 5–6 个区室间穿梭。**最常见的是额外定位到
细胞质基质或细胞核**(常见情形:预期定位到核的蛋白同时观察到基质定位,或反之)。因此:
- **工具(如 Plant-mPLoc / DeepLoc)给出多个定位往往是真实现象,不一定是"工具打架"。**
- 多定位成因复杂(时空表达变化、细胞结构动态、刺激诱导转移、修饰与互作、细胞周期依赖等),
  **单次静态预测/图像无法捕捉完整定位**。
- **常见定位先验**(可用于结果合理性核查):无论动植物,定位到**细胞核与细胞质基质**的
  蛋白最多,其次是**线粒体、质体、细胞膜**。


## 4. 实验校正与"预测/实验不符"的应对 / Experimental validation

**四种实验研究方法**(预测之外的验证手段):
- **融合报告基因定位法(GFP/GUS 等)**:目标蛋白与报告基因融合,追踪荧光信号定位——
  **目前最常用**。
- **免疫荧光标记定位法**:特异性荧光标记抗体与目标蛋白结合后检测信号。
- **亚细胞结构分离定位法**:超速离心分离细胞器后提取蛋白分析,适合蛋白质组级别,常配合
  双向电泳 + 质谱。
- **生物信息学预测**:辅助方法(见上,只能作参考)。

**细胞器 marker 识别特征**(读 GFP 共定位结果时有用):叶绿体大(~5 μm)且在特定激发光下
**自发红色荧光**,最易辨认;线粒体小圆点(~0.5 μm)、密集分布;细胞核多为圆点状、居细胞
边缘或中心(转录因子多入核,可由核信号肽 NLS 预测);内质网呈网状遍布;高尔基体表达时
内质网通常也可见;细胞膜在原生质体下为圆圈状;液泡膜为明显的胞内膜。

**当预测与实验不符、或双定位被审稿人质疑时(实战方案):**
1. **多法互证**:尽量用 ≥2 种方式验证 + 平行实验,排除方法与人为因素。
2. **获取动态结果**:分时段观察、或换不同宿主物种重复(异源表达时宿主分选受体差异可能
   导致定位偏差)。
3. **从蛋白互作解释**:互作蛋白一般共定位——已知互作蛋白定位于核,则目标蛋白可能定位
   到核或为双定位;可敲除/干扰互作蛋白后在缺失宿主中重新定位以验证。
4. **NLS/NES 生信分析**:含弱 NLS 或 NES 信号的蛋白只能部分入核/出核,出现双定位属正常。
5. **序列一致性参比**:把目标蛋白在 UniProt 中 BLAST,**一致性 >60%** 的蛋白作参比;
   若参比蛋白存在双定位,目标蛋白也可能双定位。(呼应工具清单中的 UniProtKB 交叉验证)

**论文措辞建议**:预测与实验不一致时,可结合"多重定位 + 分选信号/互作"解释,而非简单
判定某一方有误。

> 植物体系选择速记:对原生质体有毒性或需长时间表达的基因更适合**烟草叶片侵染**;
> 定位到具体细胞器、需清晰分辨的基因更适合**原生质体**(信号弱可增强启动子并重复 2–3 次)。

## 5. 命名对照(避免把"同一区室的不同叫法"读成不同结果) / Terminology cross-reference

由工具文档可直接确定的**信号类型 → 区室**对应(可放心使用):
| 信号缩写 | 含义 | 对应区室 |
|---|---|---|
| cTP | chloroplast transit peptide | 叶绿体 Chloroplast(= 质体 plastid) |
| mTP | mitochondrial targeting peptide | 线粒体 Mitochondrion |
| SP | signal peptide | 分泌途径 / 胞外 Secretory/Extracellular |
| lTP | thylakoid luminal transit peptide | 类囊体腔 Thylakoid lumen |
| PTS1 | peroxisomal targeting signal 1 | 过氧化物酶体 Peroxisome |

> 🧩【待补充】不同**工具输出标签**之间的对照——哪些命名看似不同实为同一区室
> (如各工具对"叶绿体/质体""液泡/溶酶体""细胞膜/质膜"的不同写法),以及是否存在
> "看似相同实有细分差异"的情况。这是新手最容易读错结果的地方。
